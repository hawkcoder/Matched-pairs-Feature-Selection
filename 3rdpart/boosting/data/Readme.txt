This zip archive file contains two text file (in addition to this readme file).

1. Relapse_time.txt - This file contains a listing of subject IDs and an indicator of whether the subject relapse early or late

2. mydata.txt - This file contains gene expression profiles of the 35 subjects. There are 70 columns of expression data with each consecutive pair
corresponding to gene expressions at original diagnosis and at relapse, respectively. The data consists of expression measures from 22,283 probe sets.
The order of the 35 pairs of expression data (columns) corresponds to the row order of the time to Relapse_time.txt data file.
